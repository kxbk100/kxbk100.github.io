# ES6的let、const和ES5的var的区别
- 使用var（ES5）声明的变量，其作用域为该语句所在的函数内，且存在变量提升现象
- 使用let（ES6）声明的变量，其作用域为该语句所在的代码块内，不存在变量提升
- 使用const（ES6）声明的是常量，在后面出现的代码中不能再修改该常量的值

let使得JS存在块级作用域
