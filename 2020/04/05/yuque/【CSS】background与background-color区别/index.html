    理论

    • background 可以设置图片，背景图拉伸、背景图大小、背景图相对位置、背景颜色等
    • 而 background-color 只能设置背景色

    验证

    以上代码的目的是，当鼠标停留在 a 文字上时，用蓝色替换红色背景图片，运行效果如下

    使用 background，生效

    使用 background-color，不生效

    结论

    background-color 改变的只是背景色，而不是背景图，效果是有的，只是图片把背景色遮住了看不到，所以建议改变背景颜色时，尽可能使用 background
